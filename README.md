# MvcProjeKampi
### Açıklama
Asp Net Mvc5 ile hazırlanmış admin ve yazar panelli, yazarların kategoriler
üzerinden açılan başlıklara yazabildiği sözlük projesi.
### Kullanılan Teknolojiler
- C#
- Asp.Net Mvc5
- Code First
- N Tier Architecture
- MS SQL
- Html, Css, Bootstrap
